# Employee Payroll Management System

An AI-powered Employee Payroll Management System built using Django, Bootstrap, SQLite, and JavaScript.

This project helps HR teams manage employees, attendance, payroll, salary prediction, reports, and chatbot-based employee queries.

---

# ⚙️ Installation

## 1️⃣ Clone Repository

```bash
git clone https://github.com/srashtisikarwar/Employees_Payroll_System.git
```

---

## 2️⃣ Go to Project Folder

```bash
cd Employees_Payroll_System
```

---

## 3️⃣ Create Virtual Environment

```bash
python -m venv myenv
```

---

## 4️⃣ Activate Virtual Environment

### Windows PowerShell

```bash
.\myenv\Scripts\Activate.ps1
```

### CMD

```bash
myenv\Scripts\activate
```

---

## 5️⃣ Install Requirements

```bash
pip install -r requirements.txt
```

---

## 6️⃣ Run Migrations

```bash
python manage.py makemigrations
python manage.py migrate
```

---

## 7️⃣ Run Server

```bash
python manage.py runserver
```
